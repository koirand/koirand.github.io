すべてのトラフィックをVPN経由にするかどうかの設定

Kazuki Koide

January 26, 2021

Windows

WindowsではデフォルトですべてのトラフィックがVPNを経由する。解除するには以下の設定を行う。

  1. コントロールパネル => ネットワークとインターネット => ネットワーク接続 に移動
  2. VPNを右クリックしてプロパティを選択
  3. ネットワークタブを開いて、インターネットプロトコルバージョン4(TCP/IPv4)を選択して、プロパティを選択
  4. 詳細設定を開く
  5. 「リモートネットワークでデフォルトゲートウェイを使う」のチェックを外す

MacOS

MacOSではデフォルトではVPNのアドレス宛の通信のみVPNが使用される。すべてのトラフィックをVPN経由にするには以下の設定を行う。

  1. システム環境設定 => ネットワーク に移動
  2. VPNを選択して、詳細を選択
  3. 「すべてのトラフィックをVPN接続経由で送信」にチェックを入れる
