MacOS設定

  • バッテリーアイコンを%表示に変更
  • 時計に日付を表示
  • システム環境設定
    • 一般
      • 書類を閉じるときに変更内容を保持するかどうかを確認 にチェック
    • Dock
      • Dockを左に表示
      • 「最近使ったアプリケーションをDockerに表示」をオフ
    • Mission Control
      • 「最新の使用状況に基づいて操作スペースを自動的に並び替える」をオフ
    • キーボード
      • キーリピートとリピート入力認識までの時間をMAXに
      • ショートカット -> 入力ソース を無効に
      • ショートカット -> キーボード -> 次のウィンドウを捜査対象にする を`⌘``に変更
    • トラックパッド
      • 「強めのクリックと触覚フィードバック」をオフ
    • Apple ID -> iCloud
      • Macを探す以外のチェックを解除
    • インターネットアカウント
      • Googleアカウント登録
    • アクセシビリティ
      • ポインタコントロール -> トラックパッドオプション -> 3本指のドラッグを有効
  • Finder
    • 環境設定
      • タグ
        • タグは使わないのですべて解除
      • サイドバー
        • 良く使う項目全てにチェック
        • タグは使わないので解除
      • 詳細
        • 拡張子を表示
        • 30日後にゴミ箱から項目を削除
    • メニューバーの表示
      • タブバーを表示
      • パスバーを表示

dotfiles

事前にGoogle DriveからSSH鍵をダウンロードして.sshディレクトにいれておく。

$ git clone git@github.com:koirand/dotfiles.git ~/dotfiles
$ cd ~/dotfiles
$ make install

Homebrew

$ /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install.sh)"
$ brew bundle --global

fish shell

$ sudo sh -c "echo /usr/local/bin/fish >> /etc/shells"
$ chsh -s /usr/local/bin/fish

terminfo

$ cat <<EOF >>/tmp/xterm-256color-italic.terminfo
# A xterm-256color based TERMINFO that adds the escape sequences for italic.
xterm-256color-italic|xterm with 256 colors and italic,
  sitm=\E[3m, ritm=\E[23m,
  use=xterm-256color,
EOF
$ tic /tmp/xterm-256color-italic.terminfo

Italic fonts in iTerm2, tmux, and vim - Alex Pearce
